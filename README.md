# Omar Castorena - Software Engineer Portfolio

Live site: https://ocastorena.github.io/my-portfolio

Software engineer focused on full-stack web development and firmware systems.

## About This Project

This repository contains my personal portfolio website, designed to highlight my background, skills, and selected work in a clean, responsive interface.

## Tech Stack

- Frontend: React, TypeScript, Next.js, Redux Toolkit, Tailwind CSS, shadcn/ui, Framer Motion, Three.js
- Backend: Node.js, Express, Java EE, RESTful APIs, Passport.js
- Databases: PostgreSQL, SQL Server, MongoDB, Prisma ORM
- Testing and Quality: Vitest, React Testing Library, ESLint, Prettier
- Tooling and DevOps: Git, Docker, Linux, Vite, GitHub Actions, GitHub Pages

## Key Features

- Responsive, single-page portfolio experience
- Animated hero section with Three.js/Vanta background
- Skills and project sections with smooth section navigation
- Resume download and contact call-to-action
- CI/CD pipeline with GitHub Actions and GitHub Pages deployment
